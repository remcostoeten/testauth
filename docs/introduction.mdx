# Introduction

Welcome to the documentation for our Next.js Authentication and Developer Tools Suite. This project combines modern authentication solutions with developer productivity tools, creating a robust foundation for web applications.

## What We're Building

This project is a comprehensive starter kit that includes:

1. **Authentication System**
   - Custom JWT-based authentication
   - OAuth2 integration (Google, GitHub)
   - Secure session management
   - Protected API routes

2. **Developer Tools**
   - Quick Notes Widget
     - Floating action button for quick access
     - Task and note management
     - Markdown support
     - Real-time persistence

3. **Database Integration**
   - User management
   - Notes and tasks storage
   - Relational data modeling

## Core Features

### Authentication
- Custom JWT implementation with secure cookie storage
- OAuth2 providers integration
- Session management and persistence
- Protected routes and API endpoints
- User profile management

### Quick Notes Widget
- Floating widget accessible from any page
- Task management with completion tracking
- Markdown notes with live preview
- Real-time database persistence
- User-specific data isolation

### Database Features
- User profiles and relationships
- Notes and tasks with metadata
- Automatic timestamps
- Data validation and type safety

## Tech Stack

### Core Technologies
- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Prisma** - Type-safe database ORM

### Authentication
- **JWT** - JSON Web Tokens for session management
- **OAuth2** - Third-party authentication
- **Cookies** - Secure session storage

### Database
- **SQLite** - Development database
- **Prisma Client** - Database access layer
- **Migrations** - Database version control

### UI Components
- **Radix UI** - Accessible component primitives
- **Lucide Icons** - Modern icon system
- **Tailwind CSS** - Utility-first styling
- **clsx/tailwind-merge** - Dynamic class management

### Developer Experience
- **TypeScript** - Static type checking
- **ESLint** - Code quality
- **Prettier** - Code formatting
- **React Markdown** - Markdown rendering
